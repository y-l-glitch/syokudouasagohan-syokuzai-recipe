場所: デスクトップ\Claude\recipe-search\
公開ページ: https://＜ユーザー名＞.github.io/recipe-search/

※このプロジェクトは「手動アップロード」運用。
  (shiroのくらし側はワンクリック自動化済み。こちらも自動化したくなったらClaudeに相談)


●ふだんの更新(新着動画を反映したいとき)

  1. このフォルダで「collect.py」を実行する:

       python collect.py

     (新着動画の字幕を取得して index.html が新しくなる)

  2. GitHubのリポジトリページを開く
  3. Add file → Upload files に index.html をドラッグ&ドロップ
  4. 緑の「Commit changes」ボタンを押す
  5. 1〜2分後に公開ページへ反映される


●ホーム画面のアイコン画像を変えたいとき

  1. 新しい画像を用意する
     ・PNG形式・正方形(180×180ピクセル以上)
     ・ファイル名は必ず「apple-touch-icon.png」(半角英数)にする
  2. GitHubのリポジトリページ → Add file → Upload files で
     apple-touch-icon.png をアップロード(同名なので自動上書き)
  3. iPhone側で反映させる:
     ・ホーム画面の古いアイコンを長押し→削除
     ・数分待ってからSafariでページを開き直す
     ・共有ボタン→「ホーム画面に追加」

  ※iPhoneはアイコンを強く記憶するので「一度削除して追加し直す」のがポイント


●「字幕未取得」の動画を取り直したいとき

  YouTubeの制限(429エラー)で字幕を取れなかった動画が10本ほどある。
  日を改めて、このフォルダで下のコマンドを実行すると再取得できる:

    python collect.py --retry-nosubs

  その後は上の「ふだんの更新」の手順2〜4でアップロードする。
  (字幕なし扱いの動画を全部再チェックするので30分くらいかかる)


●各ファイルの役割

  collect.py           … 字幕収集スクリプト本体
  template.html        … ページの見た目の雛形(デザイン変更はここ)
  index.json           … 収集済みデータ(消すと全動画を取り直しになるので注意)
  index.html           … 完成品(自動生成されるので直接編集しない)
